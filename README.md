Airline Operations and Revenue Analysis Using SQL
Designed and executed a structured SQL case study analyzing a simulated airline booking dataset to extract actionable business insights. Built and queried a relational schema comprising fact and dimension tables covering bookings, flights, airports, customers, and airlines.
Key analyses performed include total revenue computation using aggregation and IFNULL handling, booking volume measurement, airline-wise revenue ranking, route demand analysis via multi-table JOINs, seat class segmentation, customer spend profiling, delay performance benchmarking, and customer segment revenue breakdown.
Key findings include Vistara leading airline revenue at ₹3.8M, Economy class dominating seat demand, Mumbai–Hyderabad emerging as the top route, the Student segment generating the highest revenue at ₹4.4M, and SpiceJet recording the highest average delays.
Presented findings in a professionally designed 10-slide deck with a dark aviation theme, structured to communicate both technical SQL methodology and business outcomes clearly.
Tools: MySQL · SQL Aggregations · Business Intelligence · Problem Solving
